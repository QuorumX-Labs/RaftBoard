# 📘 RaftBoard

### Distributed Real-Time Collaborative Whiteboard with Mini-RAFT Consensus

## 🚀 Overview

RaftBoard is a fault-tolerant, distributed real-time collaborative
whiteboard system using a Mini-RAFT consensus protocol.

## 🧠 Features

-   Real-time collaboration
-   Leader election
-   Log replication
-   Fault tolerance
-   Zero-downtime reloads

## 🏗️ Architecture

Client → Gateway → Leader → Replicas

## 📁 Structure

-   gateway/
-   replica1/
-   replica2/
-   replica3/
-   frontend/
-   docker-compose.yml

## 🐳 Run

docker-compose up --build
